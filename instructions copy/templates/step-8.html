Current Weather

On index.html, use expressions and directives to build out the page below

  • Expressions in Angular serve as the equivalent to Knockout's text binding. Use the expression {{ current.name }} to pull in the name of the current city
  • Use expressions to display the temperature in Fahrenheight, the description of the current weather conditions, and an icon.
  • For the temperature, the API returns it in degrees Kelvin. Convert it into Fahrenheight using math inside the Angular expression
  • The src of the image follows the following syntax: http://openweathermap.org/img/w/IMAGEID.png
