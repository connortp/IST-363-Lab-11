Activate Angular JS

Similar to Knockout and its ko.applyBindings method, after constructing a module and controller, Angular needs to be told to start using them somewhere on the webpage.

  • On index.html, add an ng-app directive onto the <body> tag and set the value equal to the name of the module you made
  • Add a ng-controller directive and set the value equal to the name of the controller
  • You can verify Angular JS is working by writing the expression {{ 1 + 2 }} somewhere inside the body tag. If you see the number 3 appear in its place in the browser, Angular JS is working correctly. It's compiler saw the expression and evaluated the result. If you see {{ 1 + 2 }} on the page, check the browser console for any errors
