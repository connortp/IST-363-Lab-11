Module and Controller

To start using Angular JS, you will need to at least create a module and a controller. A module acts a container for all the different parts of an Angular application - here it will represent the entire weather information site. A controller is similar to a viewmodel in Knockout, but also takes on more traditional controller functionality in an MVC framework

  • In app.js, create a variable called app and declare a new angular module on it called weather
  • Create a new controller called mainCtrl on the weather module
  • Inside the main controller, create a constant called key and set it equal to the Open Weather Map API key you copied from the previous step
