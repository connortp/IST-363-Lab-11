Forecasted Weather

Request

  • Back in main.js, create another AJAX request underneath the first. This will be used to get data for a 5 day forecast with conditions every 3 hours
  • For this request:
    • Use http://api.openweathermap.org/data/2.5/forecast as the URL endpoint
    • Use the same parameters as the current weather request
    • console.log out the response returned from the API and if it's successful, set $scope.forecast equal to the data from the response

View

  • On index.html, use a Bootstrap table and ng-repeat to loop through the array of forecasted weather data points and create a table row for each
  • Put the weather icon, date & time, temperature in degrees Fahrenheight, and the condition description in separate table cells
  • For the date and time, the API returns the datetime of the weather data point as a UNIX timestamp. Convert this to a proper date format using a date filter
