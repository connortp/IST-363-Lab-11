Navbar Component

The navbar at the top of the site is a standard Bootstrap component. While this could be put directly on index.html, create a component for it so it can be reused should this site have more pages in the future

  • Create a new component on app called navbar
  • Using template, add the HTML for the bootstrap navbar in your desired choice (single string, array of joined strings, or template literal)
  • Add the <navbar> component tag onto index.html
